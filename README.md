# Nix Neovim Configuration

This project contains a custom Neovim configuration using Nix. 
It includes plugins, configurations, and scripts to enhance the development experience.

## How to Use

1. Clone this repository.
2. Make sure you have Nix installed.
3. Follow the instructions in the configuration files to customize your environment.
